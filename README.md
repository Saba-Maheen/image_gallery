# Image Gallery

A responsive Image Gallery built using HTML, CSS, and JavaScript. This project allows users to browse images, filter them by category, and view them in a lightbox with navigation controls.

## Features

- Responsive image gallery layout
- Image filtering by category
- Lightbox image viewer
- Next/Previous image navigation
- Hover effects and smooth transitions
- Mobile-friendly design
- Clean and simple user interface

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

##  How It Works

1. Open the gallery in your browser.
2. Use the filter buttons to display specific categories.
3. Click any image to open it in a lightbox.
4. Use the Previous (❮) and Next (❯) buttons to navigate between images.
5. Click the close (×) button to exit the lightbox.
